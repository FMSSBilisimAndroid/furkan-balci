## Açıklama
API üzerinden alınan mars verilerini işleyip, recycler view ile listeliyoruz ve tıklanınca bir detay ekranına gönderiyoruz.

```
https://mars.udacity.com/realestate
```
Verileri üstte bulunan adresten alıyoruz.
